07 — Design Tokens
CSS Variables cần áp dụng
Copy đoạn :root này vào tất cả 3 trang để thống nhất toàn bộ hệ thống.
Token Value Dùng cho
--color-primary #007AFF Buttons, links, focus ring, active states
--color-primary-hover #0056CC Button hover state
--color-primary-light rgba(0,122,255,.10) Secondary buttons, nav active bg, focus shadow
--color-success #34C759 Approve button, success badge
--color-warning #FF9500 Pending badge, warning toast
--color-danger #FF3B30 Reject button, error badge, destructive actions
--color-bg #F2F2F7 Page background (tất cả 3 trang)
--color-surface #FFFFFF Card, input, modal background
--color-border rgba(0,0,0,.10) Card border, divider, nav border
--color-text #000000 Primary text
--color-text-secondary rgba(60,60,67,.60) Subtext, labels, placeholders
--font-display 'Plus Jakarta Sans' h1, h2, h3, nav logo, card titles, amounts
--font-body 'DM Sans' body, labels, inputs, buttons, captions
--radius-lg 14px Cards, panels, form sections
--radius-md 10px Inputs, buttons, badges
--radius-sm 8px Small buttons, chips, tags
